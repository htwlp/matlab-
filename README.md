结构，细胞体（cell，我一般这么叫）
  结构，首先它类似一个树，
  基本结构是a.(结构名称)b(分支名称)=‘’（字符串，矩阵，随意）
  嵌套结构体：a(名称)=struct('','','结构体分支名称',struct,'',[],);
  结构体内容读取：a.b直接写分支名称，好像没有可以直接一次性导出的办
细胞体cell
  类似一个矩阵，每个矩阵的元素都是可以是数组字符串
  存储方式：a(1,1)={}(左右必出现一{})
  内容读取：a{}，直接读取a（）
  cell的每一个小块（名称加坐标）都是矩阵，可以直接被引用
  num2cell，将一个矩阵拆解成许多小矩阵
  mat2cell（a，【1，1，1】，3）（对a操作，拆解成三行三列）
    多维细胞体cat
    c=cat(1,,)   [A

    C=cat(2,,)[AB]
    c=cat(3,,)[AB]三维细胞体，引用后面的a（，，2）
